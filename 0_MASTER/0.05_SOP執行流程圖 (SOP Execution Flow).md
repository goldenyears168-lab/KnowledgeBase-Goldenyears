👥 角色協作流程圖
副攝的一天
開店前
  ├─ 環境檢查
  ├─ 系統開啟
  └─ 現金點交

客戶到來
  ├─ 客戶接待
  ├─ 預約核對
  └─ 動線分流

拍攝中（等待）
  └─ 準備結帳流程

結帳時
  ├─ 接收攝影師交接
  ├─ 收款
  ├─ 系統更新（五星標記、店內備註）
  └─ 送客

收店時
  ├─ 環境整理
  ├─ 批次上傳
  ├─ LINE 通知修圖師
  └─ 存錢與回報
攝影師的一天
拍攝前
  ├─ 器材檢查
  ├─ 燈光設定（依據產品線）
  └─ 確認客戶資訊

拍攝中
  ├─ 引導與互動（2-3分鐘）
  ├─ 拍攝執行（10-12分鐘，30-50張）
  └─ 挑圖引導（2-3分鐘）

結帳時
  ├─ 攝影師交接（攝影師 → 副攝）
  ├─ 口頭交接：選定張數 + 修圖備註
  └─ 陪同結帳

拍攝後
  └─ 器材歸位、準備下一位客戶
修圖師的一天
接單
  ├─ 查看後台資訊系統
  ├─ 確認急件優先序
  └─ 接單

修圖
  ├─ 依據修圖規範 
  ├─ 品牌標準
  └─ 各產品線差異

品質控管
  └─ 店長或指定人員檢查

更新系統
  ├─ edited photo link
  └─ 祝福小語

交件
  ├─ Email 交件
  └─ 校稿流程（如有需要）

🔄 關鍵交接點
交接點 1：彩妝師 → 攝影師
彩妝師完成
  ├─ 確認妝髮完成
  └─ 通知副攝

副攝
  └─ 引導客戶至攝影區

攝影師
  └─ 確認客戶準備就緒，開始拍攝
交接點 2：攝影師 → 副攝（關鍵！）
攝影師口頭交接：
  ├─ 「最終選定張數：X 張」
  └─ 「修圖備註：...」（如有特殊需求）

副攝確認：
  ├─ 記錄在後台資訊系統
  └─ 確認資訊無誤
交接點 3：副攝 → 修圖師
副攝（收店時）：
  ├─ 批次上傳完成
  └─ LINE 群組通知修圖師

通知格式：
  「[日期] 批次上傳完成
   共 X 位客戶，Y 張照片
   急件：Z 位
   請查看後台資訊系統接單」

修圖師：
  ├─ 查看後台資訊系統
  └─ 確認接單
⚠️ 關鍵注意事項
黃金原則（必須遵守）
1. 嚴禁在 LINE 傳遞客戶 SOP 資訊
    ◦ 所有客戶資訊必須透過「後台資訊系統」登錄
    ◦ 攝影師的「修圖備註」必須在結帳時口頭交接給副攝
2. 攝影師交接（階段 3）
    ◦ 攝影師必須在結帳時向副攝口頭交接選定張數與修圖備註
    ◦ 確保資訊傳遞無誤
3. 急件優先處理
    ◦ 急件必須在當日完成修圖並交件
    ◦ 收店時必須確認急件已通知修圖師
📊 品質檢查點
店長監督重點
• 階段 1： 開店 SOP 執行、預約核對準確性
• 階段 2： 攝影師互動品質、拍攝時間控制
• 階段 3： 攝影師交接、系統更新準確性
• 階段 4： 批次上傳完成、通知已發送
• 階段 5： 修圖品質、交件時效
