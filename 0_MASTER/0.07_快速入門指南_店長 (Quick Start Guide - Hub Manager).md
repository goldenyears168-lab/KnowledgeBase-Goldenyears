快速入門指南：店長 (Quick Start Guide - Hub Manager)
歡迎加入「好時有影」！ 這份指南幫助您在 30 分鐘內了解最重要的信息，開始您的店長之旅。
--------------------------------------------------------------------------------
🎯 您是誰？
您是「好時有影」的現場指揮官，負責確保分店營運順暢、客戶體驗優質、夥伴協作高效。
核心角色：
• ✅ SOP 執行教練（確保所有流程 100% 準確）
• ✅ 全方位管理者（營運、行銷、人才、財務）
• ✅ 客戶體驗守護者
--------------------------------------------------------------------------------
📋 第一天：必讀清單（30 分鐘）
步驟 1：理解品牌靈魂（10 分鐘）
📄 閱讀： 0_MASTER/0.01_0.01_好時有影總綱.md 的「第 1 章：我們的靈魂」
重點：
• ✅ 核心價值觀：「職涯啟程」與「祝福儀式」
• ✅ 您的使命：讓客戶「看見那個真實又美好的自己」
步驟 2：了解您的角色（10 分鐘）
📄 閱讀： 2_PARTNERS (夥伴角色規範)/2.1_分店角色 (Store Roles)/2.11_Spec_店長.md 的「第 1-2 章」
重點：
• ✅ 您的五大核心職責：SOP管理、營運管理、行銷執行、人才管理、財務管理
• ✅ 您需要協作的夥伴：副攝、攝影師、總部自動化夥伴
步驟 3：掌握核心 SOP（10 分鐘）
📄 閱讀： 1_CLIENT_PROJECTS (客戶專案SOP)/ 1.01_SOP工作室產品手冊.md 的「第 1 章：SOP 總覽」
重點：
• ✅ 五個服務階段：開店準備 → 攝影服務 → 協同結帳 → 收店上傳 → 後期製作
• ✅ 您的監督重點：階段 1（開店）、階段 3（結帳）、階段 4（收店）
--------------------------------------------------------------------------------
📚 第一週：學習路徑

第 5 天：掌握管理工具

--------------------------------------------------------------------------------
🎯 關鍵 SOP 快速參考
每日必做（Daily Checklist）
開店前：
• [ ] 檢查環境與器材（依據「📄 1.01_SOP工作室產品手冊.md」階段 1）
• [ ] 確認當日預約（查看 Simplybook）
• [ ] 檢查後台資訊系統是否有待處理項目
營運中：
• [ ] 監督副攝執行 SOP（階段 1-3）
• [ ] 協調攝影師與客戶的互動
• [ ] 監控急件處理進度
收店前：
• [ ] 確認批次上傳完成（階段 4）
• [ ] 確認 LINE 通知已發送給修圖師
• [ ] 檢查財務記錄（存錢與回報）
--------------------------------------------------------------------------------
🤝 關鍵協作關係
您（店長）
  ├── 副攝：監督 SOP 執行、培訓新人
  ├── 攝影師：協調拍攝流程、確保品質
  ├── 總部自動化夥伴：接收數據報告、協作系統改善
  └── 執行長：重大決策請示、SOP 改善建議
--------------------------------------------------------------------------------
📞 遇到問題時
SOP 執行問題
1. 先查閱 1.01_SOP工作室產品手冊.md
2. 若仍不清楚，詢問「執行長 (CEO)」
3. 記錄問題，提出 SOP 改善建議
系統技術問題
1. 先查看 3.04_SOP_資訊 (IT).md
2. 聯繫「總部自動化夥伴」協助
3. 必要時聯繫「執行長」
夥伴管理問題
1. 參考 3.02_SOP_人才 (Talent).md
2. 依據「夥伴分級制度」處理
3. 重大問題請示「執行長」
