🎯 您是誰？
您是「好時有影」的專業影像執行者與祝福儀式主理人，透過精湛的技術與真誠的互動，為客戶創造「看見那個真實又美好的自己」的影像體驗。

• ✅ 專業影像執行者（技術 + 專業）
• ✅ 祝福儀式主理人（溫度 + 互動）

🎯 關鍵 SOP 快速參考
拍攝前檢查清單（Pre-Shoot Checklist）
器材準備：
• [ ] 相機與鏡頭檢查（曝光、對焦測試）
• [ ] 燈光設備檢查（各產品線的標準設定）
• [ ] 記憶卡空間確認
客戶準備：
• [ ] 確認客戶預約資訊（後台資訊系統）
• [ ] 確認是否有彩妝師服務（動線分流）
• [ ] 準備引導話術（依據產品線）
拍攝執行流程（15 分鐘標準）
1. 引導與互動（2-3 分鐘）
   └─ 營造輕鬆氛圍、建立信任

2. 拍攝執行（10-12 分鐘）
   └─ 標準張數：30-50 張
   └─ 確保曝光準確、構圖穩定

3. 挑圖引導（2-3 分鐘）
   └─ 協助客戶挑選最滿意的照片
   └─ 記錄「修圖備註」
協同結帳流程（關鍵！）
1. 攝影師交接
   └─ 攝影師 → 副攝

2. 口頭交接內容
   └─ 「最終選定張數」
   └─ 「修圖備註」（如有特殊需求）

3. 陪同結帳
   └─ 確保資訊傳遞無誤
   └─ 送客時體現「祝福儀式」
--------------------------------------------------------------------------------
🤝 關鍵協作關係
您（攝影師）
  ├── 副攝：交接「選定張數」和「修圖備註」
  ├── 店長：協調拍攝流程、確保品質
  ├── 彩妝師（如有）：確認妝髮完成後開始拍攝
  └── 修圖師：透過副攝傳遞「修圖備註」
--------------------------------------------------------------------------------
💡 祝福儀式：互動話術範例
開場（建立信任）
"歡迎來到好時有影！我是 [您的名字]，今天會協助您完成拍攝。 不用緊張，我們慢慢來，最重要的是讓您感到放鬆和自在。"
拍攝中（鼓勵與引導）
"很好！這個表情很棒！" "放輕鬆，您現在的樣子很美。" "我們再試一個角度，您覺得怎麼樣？"
結束（祝福儀式）
"今天的拍攝完成了！您表現得非常好。 這些照片會記錄下您最真實、最美好的樣子。 期待看到最終的作品！"

📞 遇到問題時
技術問題（器材、燈光）
1. 先查閱 1.01.2_附錄_攝影技術手冊 (Photography Technical Manual).md
2. 若仍不清楚，詢問「店長 (Hub Manager)」
3. 緊急情況：使用備用器材或調整拍攝方案
客戶互動問題
1. 參考 1.01_SOP工作室產品手冊.md 的「3.2.1 拍攝互動」
2. 以「祝福儀式」的核心價值為指導原則
3. 必要時請「店長」協助
SOP 執行問題
1. 查閱 1.01_SOP工作室產品手冊.md 的相關章節
2. 若 SOP 未定義，以「核心價值觀」為決策依據
3. 記錄問題，向「店長」反映
