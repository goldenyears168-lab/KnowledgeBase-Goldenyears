1. 團案業務接洽流程
1.1 基本原則
任何確定的團案業務，就是要創群組，把相關的人都加入群組，交接或問問題比較方便。
請不要用私LINE的方式，會有溝通的問題。
好時有影沒有團體優惠方案，倒是可以想想有什麼公關合作的機會。例如想想降低到府外拍門檻，送合照或活動紀錄，或當活動贊助商，透過合作夥伴找到更多潛在顧客。顧客付費999，才會這張999的價值，有重視的感覺。經過許多經驗，發現重視優惠的顧客寧可不要。我們更要學會提升形象，想到更專業的做法。
1.2 接洽流程
一、對方看官網企業團體
有價格和說明流程
二、對方填寫團體諮詢連結
三、自動提供報價單
報價單模板：
報價單用意：確定要拍，且確定拍攝人數、時間、交件方式、匯款方式、匯款日期，才會提供報價單
四、同時寄名單模板
名單範本：
• 店內拍為三週前提供
• 外拍為一個月前提供
確定要的時候請對方匯款訂金3000元，匯款帳戶 中國信託822 帳號178540191440
如果對方提供名單，問一下裡面有沒有主管或老闆，我們不會怠慢他，我們也幫他多留一點時間拍攝
1.3 注意事項
• 如果對方不先提供名單和確切拍攝時間，我們就不能先預留時間給對方。
• 好時有影恕不回報追蹤預約與拍攝進度，請內部自行追蹤。
• 可以直接線上預約的貴賓，也不能幫他們預約。

1.5 拍攝方式
請對方預約拍攝
一律須先請對方支付款項，提供名單，於一個月內各自預約到店拍攝完畢。
可以出機拍攝嗎
• 出機要一個半月前確認確切日期、提供所有名單
• 超過20人，加價攝影費一天3000+車資實報實銷，可參考Uber的試算
• 外拍流程請參考「2. 外拍服務流程」章節
對方有特殊要求（例：裁切要裁到胸上、裁切比例要多少）

2. 外拍服務流程
2.1 外拍時間與酬勞計算
整點把攝影棚跟電腦架好，拍攝時間才開始計算外拍鐘點費。
願意搭車來回提早參與從中山店搬運器材+架棚的人，可多給搬運架棚車馬費每人200元。
下車時間為拍攝時間提早一小時，雖然20分鐘就可以架好棚器材，提早到還是比較安全。
跟顧客說整點+10分開拍
最後一組表定時間+30分鐘為挑圖的緩衝時間。
鐘點費計算： 表定的拍攝時間 - 彈性休息一小時（自己找時間輪流休息）+ 緩衝時間 + 收拾時間。
隔天要繼續拍： +30分鐘緩衝 + 30分鐘（調色上傳照片）下班
隔天不用繼續拍： +30分鐘緩衝 + 60分鐘（調色上傳照片&收拾棚）上車
外拍的重點是把顧客服務好，效率自己掌握，提早結束，提早下班。
2.2 各項補助
3. 車馬費： 每人每趟來回共500元，新竹以南（不含新竹）則為1000元
4. 住宿補助： 新竹以南（不含新竹）過夜住宿每人每日補助1000元（例如：在台中連續拍攝4天，補助3天住宿）
5. 外拍餐費： 每人每日補助150元
如果客人沒請吃飯，結束後各自填寫請款申請單報外拍餐費，報統編93712237；如果客人有請吃飯，還是有這個預算可以買飲料。
2.3 外拍規劃與人力配置
外拍成本預算上限 = 每位顧客×300元 + 額外收的交通住宿費
先依據人力預算安排出勤人數與交通住宿。如果客戶沒有額外負擔就得控制在我們預算內。
外拍之前就要寫預算表，統計累積時數，才不會每個人會分到多少不知道。
一般情況： 外勤當日人力配置2+1位（如果顧客加購化妝師則4位）
• 攝影師1： 通常是外拍長，寫外拍工作表，為對接窗口，擔任攝影師並寫祝福小語關鍵字
• 副攝1： 內外場引導顧客、挑圖、看同學分享限動、收款、場控、撰寫祝福小語
• 機動1： 攝影師或副攝疲累時換班，忙碌時協助攝影師注意頭髮衣服、協助副攝讀照片、回饋問卷、清點器材、錄Reels素材、花絮照、剪輯簡單的Reels
每次外拍建立兩個聊天室：
• 對外： 對方窗口、第二部門接洽人+第四部門外拍長+執行長
• 對內： 與該次外拍相關所有工作人員
2.4 外拍前規劃時間表
前一個月：
• 建立line對外聊天室
• 提供修圖項目在記事本
• 要求拍攝名單，付訂金
前二週：
• 確認我方會到場的工作人員並建立line對內聊天室
• 完成外拍工作表的流程表部分
• 約好計程車來回載運時間
• 訂住宿處（適用新竹以南且連續拍攝二天以上）
前一週：
• 確認拍攝名單是否還要更改
• 完成外拍工作表
• 確認有無便當，若無的話，事後請出勤夥伴各自填寫支出分配申請單
• 器材留隔夜確認要能鎖門
前一～二天：
• 名單複製在外勤待修工作表，修圖師才知道
• 外拍工作表印出來（幾人出席印幾份，但實際上拿出來使用的只要三份）

3. 活動紀錄拍攝SOP
3.1 攝影任務說明
活動照片將用於新聞稿發布與剪輯影片使用。
交件時間：
• 拍攝當天繳交為佳
• 若無法，請最晚隔天全數交件
• 拍攝當天至少交出10張代表性照片
照片命名方式：
• 資料夾命名：YYYYMMDD活動名稱
• 照片命名：YYYYMMDD活動名稱-01
3.2 必拍項目與說明
• 整場以橫式構圖為主，主持人、營運長、講者特寫建議補拍直式。
• 約一小時拍攝50張、兩小時100張以上為佳，愈多愈好。
3.3 拍攝注意事項
• 講者特寫：避免他人入鏡，必要時啟用APSC裁切。
• 簡報遠景：清楚帶到講者與簡報主標。
• 大合照：廣角18mm，必要時站高，快速引導隊形。
• 快門：靜態 1/160秒以上，頒獎與互動場景 1/250秒以上。
• 多按幾張快門，避免閉眼模糊畫面。
3.4 器材與設定指引
器材配置：
• 相機 × 1台
• 鏡頭 × 2顆：廣角（18-35mm）＋中長焦（24-70mm 或 70-200mm）
• 閃光燈 × 1支（備用）
• 記憶卡 × 2張以上
• 備用電池 × 1顆以上
• 名片、攝影背心、筆
相機設定：
• 快門速度：靜態 1/160s↑，動態互動／頒獎 1/250s↑
• ISO：自動ISO，範圍 800-3200
• 檔案格式：RAW（利於後期）
• 白平衡：自動白平衡為主，必要時調整日光／白熾光
• 對焦模式：連續對焦（AF-C / AI Servo），臉部優先
• 備份：雙卡槽同步記錄RAW檔
3.5 修圖原則
4. 篩圖

4. 贊助企劃與時程
4.1 目標
• 最低30組，理想目標是100組
• 台大畢聯會證件照一定要贊助到幹部和畢代
4.2 年度時程
• 6月： 蒐集贊助訊息
• 7,8月： 主動邀約贊助
• 9,10月： 拍攝贊助/主動蒐集特約科系
• 11,12月： 拍攝特約科系的畢業證件照
4.3 每月執行內容
6月：蒐集贊助訊息
一一打電話連繫之前的窗口，請他們提供活動側拍報告、和填寫贊助回饋表單
7,8月：主動邀約贊助
1. 在店內製作文宣品宣傳有贊助企畫
2. 一週至少3篇IG文（每週六晚上9點前）、2篇限動，增加曝光度
3. 條列名單，與在IG發送訊息
4. 店內展示
5. 辦活動（例如：KKday的客製拍攝活動）
6. 設計給各系所畢代的台大畢聯會EDM
9,10月：拍攝贊助/主動蒐集特約科系
1. 參加台大社團聯展，發DM交朋友
2. 打聽台大畢聯會新幹部與各系畢代
3. 邀請校園網紅，以社團合作名義跟他們合作
11,12月：拍攝特約科系的畢業證件照
4.4 贊助方式與回饋
畢業證件照：
• 畢代送一張
• 二人同行折100，原本798變700（平均一人350），還送合照電子折價券200元
• 滿系所60%的人數來拍，就送大外拍（圖書館前的台階）+ 小外拍
贊助結束後的追蹤：
1. 感謝他們集中一天前來拍攝，籌備很辛苦
2. 口頭提醒公關組要以每張照片代表一個版位曝光
3. 檢查是否有發文（拍攝照片後10天設定行事曆檢查該系是否有發文）
