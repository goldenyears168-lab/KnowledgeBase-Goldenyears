1.1 手冊目的：實體資產即戰略資源
• 目的 : 本手冊旨在建立一套標準流程，用於管理組織的所有「實體資產」與「後勤行政」。
• 核心理念 : 我們不將「總務」視為單純的成本支出。「總務」是「好時有影」品牌體驗的實體載體。
    ◦ 我們的「資產」（相機、燈具）是攝影夥伴專業的延伸。
    ◦ 我們的「環境」（音樂、香氛、整潔度）是「祝福儀式」價值的體現。
    ◦ 本SOP的目標，是將「實體資產」視為能產生效率、體驗與信任的戰略資源來管理。
• 核心原則 :
    ◦ 效率 : 流程簡潔，易於執行。
    ◦ 安全 : 確保夥伴與客戶的環境與資產安全。
    ◦ 穩定 : 透過資產壽命與關鍵日期管理，確保營運不中斷。
    ◦ 體驗 : 確保「門市感官維護」符合品牌標準。
1.2 適用範圍
• 本SOP適用於組織中所有經手「資產」、「物料」、「環境」、「租約」的夥伴，包含執行長 、自動化夥伴 、店長 、副攝 。
1.3 權責單位
• SOP 擁有者 : 執行長 。
• SOP 執行者 : 店長  (負責監督、盤點)、副攝  (負責日常執行)。
2. 資產管理SOP 
本章節詳述組織中所有「高價值實體資產」（例如：相機、鏡頭、燈具、電腦、空調等）的標準管理流程。
2.1 資產採購SOP
• 目標： 確保所有高價值資產的採購，都符合組織的需求、預算、與審批流程。
• SOP：
    1. 需求提出： 店長  或 執行長  根據「資產清冊」(見 2.2) 或「新服務需求」（來自 SOP_研發）提出採購需求。
    2. 評估與核准：
        ▪ 需求單提交至 執行長 。
        ▪ 採購決策必須符合「📄 3.03_SOP_財務 .md」中定義的「層級授權表」。
    3. 執行採購： 由 執行長  或指定人員執行採購。
    4. 登錄資產： (見 2.2) 採購完成後，必須在 24 小時內登錄至「資產清冊」。
2.2 資產清冊SOP
• 目的： 建立組織所有資產的簡單清單，以便管理與預算規劃。
• 工具： 簡單的 Google Sheet 清單
• SOP：
    1. 資產登錄 ：
        ▪ 執行者： 執行長 。
        ▪ 時機： 新資產採購完成 24 小時內。
        ▪ 必填欄位：
            • 資產編號 (例如：CAM-2025-01)
            • 資產名稱 (例如：Sony A7IV)
            • 購入日期
            • 購入金額
            • 存放地點 (例如：公館店)
            • 保管人 (例如：店長 )
            • 預估壽命 (月) (例如：36 個月)
    2. 資產狀況記錄 ：
        ▪ 簡單記錄資產使用狀況，需要汰換時標記。
    3. 維修記錄 ：
        ▪ (見 2.4) 任何維修紀錄必須回填至此清冊。
    4. 季度審核 ：
        ▪ 執行長  需在「資訊治理季度檢討會議」中，一併審核「資產清冊」中需要汰換的項目，並提前納入下一季的採購預算。
2.3 資產盤點SOP
• 目標： 確保「帳」(資產清冊) 與「實」(現場資產) 100% 一致。
• 執行者： 店長 
• 頻率： 每月 1 號 (或指定日期)
• SOP：
    1. 店長  下載「資產清冊」中「存放地點」為該分店的列表。
    2. 逐一核對現場資產的「數量」與「狀態」（是否完好）。
    3. 在「資產清冊」中更新「最後盤點日期」與「狀態備註」。
    4. 若發現盤虧或損壞，立即依 2.4 流程回報。
2.4 資產報修與除帳SOP
• 目標： 快速反應資產故障，並確保報廢資產的帳務清晰。
• SOP：
    1. 故障回報： 副攝  或 攝影師  發現資產故障，立即通報 店長 。
    2. 啟動報修： 店長  立即聯繫「廠商聯絡清單」進行報修，並在「資產清冊」中將該資產狀態標記為「維修中」。
    3. 記錄維修： 維修完成後，將「維修日期」、「維修內容」、「維修金額」登錄回「資產清冊」的「維修記錄」欄位。
    4. 資產除帳 ：
        ▪ 觸發： 資產確認「無法修復」或「已達汰換標準」。
        ▪ 行動： 店長  提交「資產報廢單」。
        ▪ 審批： 執行長  批准。
        ▪ 執行： 自動化夥伴  (或 執行長 ) 在「資產清冊」中將該資產標記為「已除帳」，並通知 記帳士夥伴 進行相應的財務處理。
3. 物料與環境SOP 
本章節詳述組織中「非資產類物料」的採購，以及「實體環境」的維護標準。本章節的目標是確保所有分店的「品牌五感」體驗100% 一致且到位。
3.1 物料採購SOP 
• 目標： 確保營運所需的常備物料不斷供，且成本可控。
• 執行者： 副攝  (回報庫存)、店長  (執行採購與核銷)。
• SOP：
    1. 安全庫存標準：
        ▪ 店長  需依據「附錄 6.x：各分店安全庫存標準表」維護物料庫存。
        ▪ (範例) [公館店] 飲用水 x 2箱、化妝棉 x 5包、衛生紙 x 20包、咖啡豆 x 2包...
    2. 庫存回報：
        ▪ 副攝  在執行「開店/收店流程」時，若發現任何物料低於「安全庫存」，必須立即回報 店長 。
    3. 採購申請與核銷SOP：
        ▪ 店長  根據庫存回報，執行採購。
        ▪ (關鍵) 採購金額必須符合「📄 3.03_SOP_財務 .gdoc」中定義的「層級授權表」（例如：NT$ 5,000 以下可自行決定）。
        ▪ 採購後，店長  需收集憑證（發票/收據），並依照「SOP_財務」的請款流程提交核銷。
3.2 關鍵SOP：門市感官維護清單 
• 目的： 將「好時有影」的「溫度 」與「祝福儀式 」價值觀，轉化為可被檢查與維護的「五感」SOP。
• 執行者： 店長  (負責每月 1 號檢查並回報)
• 工具： 門市感官維護清單 (Google Form / Sheet)
• SOP：
    ◦ 店長  需每月 1 號，使用此清單對分店進行全面檢查，並將結果回報給 執行長 。
• 五感檢查標準：
    ◦ 1. 視覺 ：
        ▪ [ ] 背景紙：是否潔淨無汙損、無明顯皺褶？
        ▪ [ ] 鏡面：接待區、化妝區、廁所鏡面是否明亮無指紋？
        ▪ [ ] 燈光：所有燈具是否正常運作？色溫是否一致？
        ▪ [ ] 陳列：作品展示架、文宣品是否整齊歸位？
        ▪ [ ] 地板/桌面：是否無明顯灰塵或污漬？
    ◦ 2. 聽覺 ：
        ▪ [ ] 音樂：背景音樂是否為「品牌指定曲風」？
        ▪ [ ] 音量：音量是否保持在「舒適且不打擾交談」的標準分貝？
        ▪ [ ] 噪音：空調或設備是否有異常噪音？
    ◦ 3. 嗅覺 ：
        ▪ [ ] 空氣品質：店內是否無異味（食物、潮濕、汗味）？
        ▪ [ ] 香氛：(若有使用) 品牌指定香氛是否一致且濃度適中？
        ▪ [ ] 廁所：是否保持潔淨無異味？
    ◦ 4. 觸覺 ：
        ▪ [ ] 座椅/沙發：客戶等候區的座椅布料/皮革是否乾淨無破損？
        ▪ [ ] 拖鞋：室內拖鞋是否清潔、乾燥、無異味？
        ▪ [ ] 道具/布料：(若有) 供客戶使用的道具或布料是否乾淨？
    ◦ 5. 空間溫度 ：
        ▪ [ ] 空調：是否維持在舒適溫度（例如 24-26°C）？
        ▪ [ ] 燈具熱源：攝影棚燈具的熱源是否過高影響客戶？
3.3 每日/定期 清潔SOP
• 每日清潔：
    ◦ 執行者： 副攝 
    ◦ SOP： 嚴格執行 SOP_工作室  中的「2.1 開店流程」與「5.1 收店流程」的環境整理SOP。
• 定期深度清潔：
    ◦ 執行者： 店長  (負責安排)
    ◦ SOP： 依據「關鍵日期清單」(見 4.1)，安排每季/每半年的空調濾網清潔、除塵、或委外清潔服務。
4. 租約與法務SOP 
本章節詳述組織中「法務合約」與「法規遵循」的標準管理流程。本章節的目標是建立一個「風險雷達」，制度化地防範與管理營運風險。
4.1 關鍵日期管理SOP
• 目的： 記錄「租約到期」、「法規年檢」等重要日期，確保不遺漏。
• 工具： 簡單的 Google Sheet 或 Google Calendar
• SOP：
    1. 權責單位： 執行長  負責維護此清單。
    2. 登錄SOP： 任何新的「合約」或「法規要求」成立時，必須在 48 小時內登錄至此清單。
    3. 追蹤項目 (範例)：
        ▪ 類別： 租約
        ▪ 項目： 中山店租約
        ▪ 關鍵日期： 2026/08/01 (到期日)
        ▪ 提醒人： 執行長 
        ▪ SOP： 必須設定 Google Calendar 行事曆 於 到期前 3 個月 (即 2026/05/01) 發出 洽談續約 提醒
        ▪ 類別： 法規
        ▪ 項目： 公館店消防安檢
        ▪ 關鍵日期： 每年 9 月
        ▪ 提醒人： 店長 
        ▪ SOP： 提醒店長安排檢查，並將「合格報告」歸檔至指定雲端資料夾。
        ▪ 類別： 稅務
        ▪ 項目： 營業稅申報
        ▪ 關鍵日期： 每單數月 15 號
        ▪ 提醒人： 自動化夥伴  / 記帳士夥伴
        ▪ SOP： 確保SOP_財務手冊中定義的SOP已執行。
    4. 審核SOP： 執行長  需在「季度檢討會議」中，一併審核此清單，確認未來一季的所有關鍵日期皆已在掌控中。
4.2 營業登記與證照SOP
• 目的： 確保所有分店的營運 100% 合規。
• 執行者： 執行長 
• SOP：
    1. 文件歸檔： 所有分店的「營業登記」、「稅籍登記」、「消防安檢合格證」等法規文件，必須掃描並備份至「好時有影總部 」的指定資料夾。
    2. 變更SOP： 若營業地址、項目、負責人有任何變更，執行長 必須立即啟動變更登記流程。
    3. 法規遵循： 店長  負責確保現場營運（例如：消防通道、安全出口）符合「消防安檢」的標準。
4.3 合約歸檔SOP
• 目的： 集中管理所有具法律效力的合約。
• 執行者： 執行長 
• SOP：
    1. 適用範圍： 包含但不限於「店面租約」、「廠商合約」、「合作協議 」、「夥伴承攬契約」等。
    2. 契約模板位置：
        ▪ 夥伴契約模板： 📁 3.09_SOP_法務 /3.9.1_夥伴契約模板 /
            • 📄 攝影師承攬契約書.md
            • (其他夥伴契約模板)
        ▪ 其他契約模板： 📁 3.09_SOP_法務 /3.9.2_其他契約模板/（未來擴充）
    3. 歸檔流程：
        ▪ 紙本： 所有合約正本需集中存放於指定之「實體保險櫃」或「文件櫃」。
        ▪ 電子： 所有合約必須掃描為 書面通知 檔，並依照「[合約類型][對象][簽署日期]」的命名規範，儲存至「好時有影總部 」的「法務合約」專屬資料夾。
    4. 登錄： 合約簽署後，其「關鍵日期」必須同步登錄至「關鍵日期清單」(見 4.1)。
